# Linux.do Post Link Opener

一个 Chrome 浏览器扩展，用于在后台标签页中打开 linux.do 论坛的帖子链接。

## 功能

- 点击帖子链接时自动在后台新标签页打开，不打断当前浏览
- 新标签页紧跟在当前标签页后面，方便按顺序阅读
- 智能识别同一帖子内的楼层跳转，保持正常跳转行为
- 仅对帖子链接生效（`/t/xxx` 格式），不影响其他链接

## 安装

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用

安装后访问 [linux.do](https://linux.do)，点击任意帖子链接即可在后台标签页打开。

## 文件说明

- `manifest.json` - 扩展配置文件
- `content.js` - 页面内容脚本，监听点击事件
- `background.js` - 后台服务脚本，处理标签页创建
